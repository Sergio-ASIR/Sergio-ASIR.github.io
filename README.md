Proyecto Web - Clean & Go

- Descripción

Este proyecto consiste en el desarrollo de una página web para una lavandería autoservicio llamada "Clean & Go", utilizando HTML5 y CSS.

- Estructura del proyecto

El proyecto está compuesto por los siguientes archivos y carpetas:

index.html → Página principal
info.html → Página de información
contacto.html → Página de contacto con formulario
css/estilos.css → Hoja de estilos
img/ → Carpeta con imágenes del proyecto
